# NatLust

NatLust is a World of Warcraft Retail addon that plays a custom sound and shows a custom texture when the player gains a lust buff.

## Install

1. Copy `NatLust` to:
   `World of Warcraft/_retail_/Interface/AddOns/`
2. Put your texture and audio files in:
   `World of Warcraft/_retail_/Interface/AddOns/NatLust/Media/`
3. Reload the UI with `/reload`.

## Media

- Texture files go in `NatLust/Media/`
- Sound files go in `NatLust/Media/`
- The addon reads them as:
  `Interface\\AddOns\\NatLust\\Media\\<filename>`

If you use the settings panel, enter file names only, such as:
- `pedro.tga`
- `pedro.mp3`

## Audio

WoW custom audio is most reliable with:
- `MP3`, `OGG`, or `WAV`
- `44.1 kHz`
- `128/192 kbps` for MP3
- Stereo
